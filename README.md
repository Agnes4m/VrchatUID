<h1 align = "center">VrChatUID</h1>
<h4 align = "center">基于gsuid-core的VRChat查询Bot插件</h4>

> ⚠️ **项目正在维护中，部分功能尚未完成**

## 丨安装提醒

> **注意：该插件为[早柚核心(gsuid_core)](https://github.com/Genshin-bots/gsuid_core)的扩展**

## 丨安装方式

1. 发送`core安装插件VrChatUID`
2. 发送`core重启`应用插件

## 丨功能（开发中）

### 用户功能

- [x] vrc登录 - 登录VRChat账号
- [x] vrc好友 - 查看好友列表
- [x] vrc搜索世界 - 搜索VRChat世界
- [x] vrc我的信息 - 查看已登录账号信息
- [x] vrc帮助 - 显示帮助信息

### 管理员功能

- [ ] Web控制台管理VRChat绑定数据

## 丨使用方式

1. 安装插件
2. 发送`vrc登录 用户名 密码`登录VRChat账号
3. 发送`vrc帮助`查看所有可用命令

## 丨其他

- 本项目仅供学习使用，请勿用于商业用途
- 数据来源于VRChat API
